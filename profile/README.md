# Wahon Labs

Open-source tools for stable, cross-platform manga reading.

## Core Repositories

- [Wahon](https://github.com/wahonlabs/Wahon)  
  Kotlin Multiplatform app (Android + iOS) with extension-based source runtime.

- [.github](https://github.com/wahonlabs/.github)  
  Organization profile, contribution defaults, and shared community policies.
